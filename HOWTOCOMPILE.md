# COMO COMPILAR
Antes de compilar é necessário [exportar](https://github.com/UTFPR-SL/grupo-3/blob/main/EXPORTTOIDE.md), e o processo é o mesmo do Windows tanto para o Linux (Usando a IDE [CodeBlocks](https://www.codeblocks.org/)), tanto para compilar quanto pra executar.
Esse processo é válido tanto para o arquivo de teste, quanto para o arquivo main.c que calcula a mantissa.

## Windows e Linux
1. Após exportar o novo projeto, clique no ícone de engrenagem (também chamado de *Build*).
![](https://github.com/UTFPR-SL/grupo-3/blob/main/images/16.png)

2. Veja o resultado na parte **Build log**.
![](https://github.com/UTFPR-SL/grupo-3/blob/main/images/18.png)

## Linux (compilar usando terminal)

1. Abra o terminal

2. Acesse o diretório onde foi clonado
```
$ git clone https://github.com/UTFPR-SL/grupo-3.git
$ cd grupo-3
```

3. Digite o comando abaixo:
```
$ make compile
```
