# grupo-3
## Mantissa
Dado um número pelo usuário real ou inteiro, podendo ser positivo ou negativo, é calculado a caracterísitca e a mantissa.

**Exemplo: **

**número**: 318278312.43

**característica**: 318278312

**mantissa**: 43
